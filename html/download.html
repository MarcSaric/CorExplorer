Downloads

If you would like to run your own CorEx analysis, the code is available on github, along with instructions for the parameters we used in the BMC paper. The scripts for generating annotations on this website are also available on github. If you need access immediately please use the contact email. We can also provide links to the raw data from the CorEx analysis on this website if there is interest.

The data that we used all comes from the Genomics Data Commons. Data was downloaded using the following steps:

  1. Go to portal.gdc.cancer.gov to download data.
  2. I clicked on "exploration" then under the "cases" sidebar I chose TCGA-SKCM
  3. On the right frame, I chose "view files in repository"
  4. Under the "files" sidebar, I chose "Workflow type": HTSeq - FPKM-UQ
  5. On the right side, I chose "add all files to cart"
  6. In the cart, I clicked "Metadata" to download metadata and "Download" to get the sequence files.
